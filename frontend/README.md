# Freight AI Dashboard

React + Tailwind dashboard for the freight bill validation backend.

## Run

```bash
npm install
npm run dev
```

The app expects the FastAPI backend on `http://localhost:8000` and proxies API calls through `/api`.

Bootstrap login:

- username: `admin`
- password: `admin`
- tenant: `default`

## Screens

- Login / signup
- Overview command center
- Company setup
- CA partner setup
- Light and dark theme
- Demo data controls
